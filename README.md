# Warehouse CLI

**Lightweight, plugin-based warehouse management CLI — built with Clean Architecture**

```bash
pip install warehouse-cli
```

<img src="https://github.com/PyRz-Tech/WerehouseCLI/raw/main/demo.gif" alt="Demo in action" weight=500>

## Features

- Full Clean Architecture (domain → cli → infrastructure)
- Dynamic plugin system — just drop a `.py` file in `plugins/` folder
- SQLite persistence (`warehouse.db`) — no server needed
- Comprehensive test suite (all critical paths covered)
- GitHub Actions CI/CD → auto-publish to PyPI on every tag

## Quick Commands

```bash
# Add a product
warehouse add

# List all products
warehouse list

# Search products
warehouse search

# Remove product by ID
warehouse delete

# Show Totals
warehouse summary
```

## Installation

```bash
pip install warehouse-cli
```

## Development

```bash
git clone https://github.com/PyRz-Tech/WerehouseCLI.git
cd WerehouseCLI
pip install -e .
warehouse --help
```
